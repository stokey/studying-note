# Android Studio Tips

+ 为TextView在预览情况下设置值，正式运行情况下不显示
	+ 根布局添加`xmlns:tools="http://schemas.android.com/tools`
	+ 在TextView中使用`tools:something="value"`属性 

+ 快速添加`String`字符串或者`dimen`数值：`选中该值，按下option+enter键选择Extract String/Dimension Resource`

+ Text/Design模式快速切换：`Control + Shift + left/right`

+ 大写为首单词之间快速切换(`Command + left/right`)：`Editor --> General --> Smart Keys --> CamelHumps`

+ Close Others：`按住option，鼠标点击要保存的文件的关闭按钮`

+ 以新窗口的形式打开当前文件：`shift + F4`  

+ 查看可重载方法：`Command + O`

+ Java正则表达式：`Alt + Enter --> check regexp`

+ 切换文件：`Control + Tab`
